Git

  • Enter your Git notes here
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git beanch: this command lists all local branches in the current repository
  • git branch [enter branch name]: create a new branch
  • git push origin main: sends commited changes of main branch to remote repository
  • git pull origin main: fetch and merge changes on the remote server to the working directory
  • git add -A: commit all files in the working brnach
  • git diff: shows uncommited changes between the working directory and index (Q+Enter to exit git diff in command line)
